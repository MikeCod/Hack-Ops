Command Injection

When does it appear ?

Command injection attacks are possible when an application passes unsafe user supplied data (forms, cookies, HTTP headers etc.) to a system shell. 

$ssh->exec("ping -W 250 -c 4 ".$_POST['host'])

What's the risk ?

Command injection is an attack in which the goal is execution of arbitrary commands on the host operating system via a vulnerable application.

How does the exploitation work ?

0|whoami

How to patch it ?

Ideally, a developer should use existing API for their language. For example (Java): Rather than use Runtime.exec() to issue a ‘mail’ command, use the available Java API located at javax.mail.*. If no such available API exists, the developer should scrub all input for malicious characters. Implementing a positive security model would be most efficient. Typically, it is much easier to define the legal characters than the illegal characters.
